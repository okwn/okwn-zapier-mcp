# Contributing to Zapier MCP

Thank you for your interest in contributing to Zapier MCP!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/okwn/zapier-mcp.git`
3. Install dependencies: `npm install`
4. Create a feature branch: `git checkout -b feature/your-feature-name`

## Development

```bash
# Run tests
npm test

# Run linter
npm run lint

# Build
npm run build
```

## Pull Request Process

1. Ensure all tests pass and linter is clean
2. Update documentation for any API changes
3. Include a clear description of changes
4. Reference any related issues

## Code Standards

- Follow existing code style
- Write meaningful commit messages
- Add tests for new functionality
- Keep changes focused and atomic

## Reporting Issues

Please report issues with detailed information including:
- Environment details
- Steps to reproduce
- Expected vs actual behavior
